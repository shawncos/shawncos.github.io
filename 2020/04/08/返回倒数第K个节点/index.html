实现一种算法，找出单向链表中倒数第 k 个节点。返回该节点的值。
输入： 1->2->3->4->5 和 k = 2
输出： 4

自己的思路:遍历一遍链表，同时缓存所有的值。


    /**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode(int x) { val = x; }
 * }
 */
class Solution {
    public int kthToLast(ListNode head, int k) {
        List<Integer> list=new ArrayList<>();
            int len=1;
            ListNode temp=head;
            list.add(temp.val);
            while(temp!=null){
                list.add(temp.val);
                temp=temp.next;
                len++;
            }
            int count=len-k;
            return list.get(count);
    }
}

双指针解法：

  • 初始时，两个指针均指向 head。
  • 先将 q 向后移动 k 次。此时p，q的距离为 k。
  • 同时移动p,q，直到q指向nullptr,此时p->val即为答案

/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode(int x) { val = x; }
 * }
 */
class Solution {
    public int kthToLast(ListNode head, int k) {
        ListNode p = head;
        for(int i=0;i<k;i++){
          p = p.next;  
        }

        while(p != null){
            p = p.next;
            head = head.next;
        }

        return head.val;
    }
}
